사이트 계정 관리 도구

1. "사이트계정관리.exe"를 더블클릭합니다.
2. 처음 실행하면 앱에서 사용할 아이디와 8자 이상의 마스터 비밀번호로 회원가입합니다.
3. 다음 실행부터는 가입한 정보로 로그인합니다.
4. "+ 새 사이트"를 누르고 사이트명, 로그인 주소, 아이디, 비밀번호, 추가 정보를 입력합니다.
5. "저장"을 누르면 왼쪽 목록에 사이트가 추가됩니다.
6. 필요한 항목 옆의 "복사" 버튼을 누른 뒤 사이트 입력칸에서 Ctrl+V로 붙여넣습니다.
7. Chrome, Edge, Whale 중 원하는 브라우저의 열기 버튼을 누르면 로그인 주소가 열립니다.
8. 저장된 사이트는 왼쪽 목록에서 선택해 수정하거나 삭제할 수 있습니다.

보안 안내
- 저장된 값은 현재 Windows 사용자 계정에서만 해독됩니다.
- 복사한 값은 20초 뒤 클립보드에서 자동으로 지워집니다.
- 프로그램을 닫을 때도 클립보드를 지웁니다.
- 계정 데이터는 Windows 사용자 폴더에 암호화하여 저장됩니다.
- 마스터 비밀번호 원문은 저장하지 않습니다. 잊어버리면 현재 버전에서는 복구할 수 없습니다.
